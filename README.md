# CodeAlpha AWS CI/CD DevOps Project

## Overview

This project demonstrates a complete CI/CD pipeline for deploying a containerized web application using AWS and GitHub Actions.

The application is packaged using Docker, stored in Amazon Elastic Container Registry (ECR), and automatically deployed to an AWS EC2 instance.

## CI/CD Workflow

1. Developer pushes code to GitHub
2. GitHub Actions triggers automatically
3. Docker image is built
4. Image is pushed to Amazon ECR
5. EC2 pulls the latest image
6. Docker container is restarted
7. Application is deployed automatically
